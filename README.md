# gatsby-source-pirate

> A gatsby plugin to source the pirate api

## How to install

`npm install gatsby-source-pirate`

or

`yarn add gatsby-source-pirate`

## How to use

```
module.exports = {
  plugins: [
    `gatsby-source-pirate`
  ],
}
```
